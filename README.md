# 电商用户行为分析报告

基于 **10,218,360 条真实用户行为记录**的深度数据挖掘与分析看板。

## 项目概述

对某电商平台 **100,000 名用户** 在 **157 万件商品**、**7,991 个类目**上的浏览、收藏、加购、购买行为进行全链路分析，最终输出包含 **12 张可视化图表** 的 HTML 交互看板和 **6 条可落地的运营策略建议**。

## 技术栈

| 领域 | 工具 |
|---|---|
| **数据处理** | Python 3.14, Pandas, NumPy |
| **可视化** | Matplotlib, Seaborn |
| **数据清洗** | 空值检测、重复值去重、时间戳异常过滤 |
| **分析维度** | 转化漏斗、用户分层、类目分析、时段分析 |
| **交付形式** | 交互式 HTML 看板 + GitHub Pages 在线报告 |

## 核心发现

| 指标 | 数值 |
|---|---|
| 总记录数 | 10,218,360 条 |
| 用户数 | 99,999 人 |
| 浏览→加购转化率 | **75.7%** |
| **加购→购买转化率** | **90.08%** |
| 浏览→购买转化率 | 68.2% |
| 日活跃峰值 | 13:00（午间高峰） |

### 关键洞察
- **真实漏斗并非线性**：大量用户跳过「收藏」环节直接加购，传统 PV→FAV→CART→BUY 模型不适用
- **加购是最强转化信号**：加购用户中 90% 最终完成购买
- **类目购买率差异达 3 倍**：高转化类目(7.7%) vs 低转化类目(2.7%)
- **清晨 6:00 转化最高**：此时段购买转化率达 15.7%

## 数据来源

数据集包含 2017 年 9 月至 12 月约 **1021 万条**用户购物行为记录，涵盖 5 个字段：用户 ID、商品 ID、类目 ID、行为类型（pv/fav/cart/buy）、时间戳。
